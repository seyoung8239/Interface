# web14-interface: 화상면접스터디 플랫폼

## 📑 주요 기능
- WebRTC를 이용한 실시간 화상 회의
- 면접 화면 녹화 및 피드백 실시간 작성
- 녹화된 면접 영상과 피드백 작성 시점 동기화
- 작성된 피드백을 익명으로 제공
- 받은 피드백을 영상과 함께 저장, 조회 및 열람
