### 📑 주요 기능 소개

1. **실시간 면접 피드백 기능과 동기화**  
  면접을 진행하며 실시간으로 피드백을 남길 수 있고 피드백을 남긴 시점과 영상을 동기화할 수 있습니다.

2. **인터뷰 기록 저장**  
  진행한 인터뷰의 영상과, 면접에 대한 피드백들을 저장합니다.

3. **인터뷰 기록 조회**  
  진행한 인터뷰의 기록 영상과 피드백을 조회할 수 있습니다.

4. **익명 기능**  
  면접에 대한 모든 피드백은 익명으로 관리되어 조금 더 솔직한 피드백을 전달할 수 있습니다.

5. **OAuth 로그인**  
  Naver, Kakao 두 가지의 social 회원 가입을 할 수 있습니다.
